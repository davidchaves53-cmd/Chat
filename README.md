# Easy Automation Chatbot

Easy Automation is a hybrid AI-powered assistant designed for **cannabis dispensaries**, **growers**, and **business operators**, but it can also be tailored into a powerful **personal assistant** for everyday automation.

This bot uses:
- Smart AI decision-making
- Web scraping (when enabled)
- CSV-based knowledge files
- Self-learning patterns
- Local automation tasks

It is built to be flexible, fast, and capable of adapting to different industries or personal workflows.

---

## 🌿 Features for Cannabis Dispensaries
- Strain lookup and recommendations  
- Effects, flavors, terpene profiles  
- Medical-use suggestions  
- CSV-based strain database  
- Automated responses for customers  
- Can be customized for your menu or POS system  

---

## 🤖 Features for Personal Assistant Mode
- Task automation  
- Web data extraction  
- CSV processing  
- Smart memory  
- Self-learning behavior  
- Local Windows automation  

---

## 🧠 How It Works
The bot combines:
- AI reasoning  
- Local Python automation  
- CSV knowledge files  
- Optional scraping modules  
- A self-learning engine that improves over time  

It can run fully offline (CSV mode) or hybrid (web + CSV).
